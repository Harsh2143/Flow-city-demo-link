# FlowCity – Predictive Crowd Intelligence

FlowCity is an AI-powered crowd intelligence dashboard designed to monitor and predict crowd density at major public events such as festivals, concerts, sports matches, and religious gatherings.

The system helps **citizens, authorities, and event organizers** make safer and smarter decisions by providing real-time crowd insights and future crowd predictions.

## Features

- Live crowd density monitoring
- AI-powered crowd prediction using ML models (XGBoost)
- Interactive map with event locations
- 2-hour crowd forecast visualization
- Stampede risk alerts
- Smart parking suggestions
- Alternative event recommendations
- Live city footfall counter
- Multi-role dashboard:
  - Citizen View
  - Authority View
  - Organizer View

## Tech Stack

- HTML
- CSS
- JavaScript
- Leaflet.js (Interactive Maps)
- Machine Learning concept: XGBoost for crowd prediction

## Use Cases

- Festival crowd management
- Smart city crowd monitoring
- Public safety and disaster prevention
- Event planning and crowd distribution
